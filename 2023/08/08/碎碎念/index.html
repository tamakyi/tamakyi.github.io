###2.archlinux 服务器访问慢

Server = http://de3.mirror.archlinuxarm.org/$arch/$repo

###3.pikvm的登陆密码

username=admin
password=admin

####修改网页端密码的方式为

kvmd-htpasswd set admin

####SSH默认账号密码为

username=root
password=root

####VMWare相关脚本
注:$install path为vmware安装路径，$vmx path为vmx文件存放路径，$vmx name为vmx文件名称。
#####1、系统自动启动VMware虚拟机

"$install path\vmrun.exe" -T ws start "$vmx path\$vmx name"

#####2、系统自动关闭VMware虚拟机

"$install path\vmrun.exe" -T ws stop "$vmx path\$vmx name" soft

#####3、系统自动挂起VMware虚拟机

"$install path\vmrun.exe" –T ws suspend "D:\system\Windows Server 2003 Enterprise Edition\Windows Server 2003 Enterprise Edition.vmx"

#####4、系统自动重启VMware虚拟机

"$install path\vmrun.exe" -T ws reset "$vmx path\$vmx name"

#####5、系统自动暂停VMware虚拟机

"$install path\vmrun.exe" -T ws pause "$vmx path\$vmx name"

#####6、系统自动暂停恢复VMware虚拟机

"$install path\vmrun.exe" -T ws unpause "$vmx path\$vmx name"
